# 📜 README - Etiquetas HTML

## 🌍 ¿Qué es HTML y para qué sirve?
HTML (HyperText Markup Language) es el esqueleto de todas las páginas web. 🏗️ Define la estructura de un sitio con etiquetas, como si fueran bloques de construcción. 🧱

---

## 📌 Estructura básica de un documento HTML
```html
<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>🌟 Mi Página Web</title>
</head>
<body>
    <h1>📢 Encabezado Principal</h1>
    <p>📝 Este es un párrafo de ejemplo.</p>
</body>
</html>
``
---

## 🏷️ Principales Etiquetas HTML

### 1️⃣ **Etiquetas de estructura** 🏗️
- `<html>` → 📄 Indica que es un documento HTML.
- `<head>` → 🎩 Contiene metadatos, enlaces a estilos y scripts.
- `<body>` → 💡 Contiene el contenido visible de la página.

### 2️⃣ **Etiquetas de encabezados** 📰
- `<h1>` a `<h6>` → 📢 Encabezados, donde `<h1>` es el más importante y `<h6>` el menos.

### 3️⃣ **Etiquetas de texto** ✍️
- `<p>` → 📜 Párrafo de texto.
- `<strong>` → 💪 Texto en negrita.
- `<em>` → 🎭 Texto en cursiva.
- `<br>` → 🔄 Salto de línea.
- `<hr>` → 📏 Línea horizontal.

### 4️⃣ **Etiquetas de enlaces** 🔗
- `<a href="URL">Texto del enlace</a>` → 🚀 Crea un enlace a otra página.

### 5️⃣ **Etiquetas de imágenes** 🖼️
- `<img src="imagen.jpg" alt="Descripción">` → 🌅 Inserta una imagen.

### 6️⃣ **Etiquetas de listas** 📋
- `<ul>` → 📌 Lista desordenada.
- `<ol>` → 🔢 Lista ordenada.
- `<li>` → 📍 Elemento de la lista.

### 7️⃣ **Etiquetas de tablas** 📊
- `<table>` → 📑 Crea una tabla.
- `<tr>` → 🔲 Fila de la tabla.
- `<td>` → 🔳 Celda de datos.
- `<th>` → 🔠 Celda de encabezado.

### 8️⃣ **Etiquetas de formularios** 📝 
> Las veremos más adelante en el curso
- `<form>` → 📥 Crea un formulario.
- `<input>` → ✏️ Campo de entrada.
- `<textarea>` → 🗒️ Área de texto.
- `<button>` → 🔘 Botón interactivo.
- `<select>` y `<option>` → 📑 Menú desplegable.

### 9️⃣ **Etiquetas semánticas** 🎭
- `<header>` → 🏡 Encabezado de una sección o página.
- `<nav>` → 🧭 Barra de navegación.
- `<section>` → 📦 Sección de contenido.
- `<article>` → 📰 Contenido independiente.
- `<aside>` → 🗂️ Contenido relacionado.
- `<footer>` → 👣 Pie de página.

### 🔟 **Etiquetas multimedia** 🎵🎥
- `<audio>` → 🎧 Agregar audio.
- `<video>` → 📹 Insertar videos.
- `<iframe>` → 🌍 Incrustar contenido de otras páginas.
